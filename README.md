# 📊 Retail Data Analysis Case Study

## 📌 Project Overview
This project focuses on analyzing retail transaction data by performing data auditing, cleaning, and generating business insights using SQL and Power BI.

## 🛠️ Tools & Technologies
- SQL (MySQL / SQL Server)
- Power BI
- Excel (for initial data review)

## 📂 Dataset
The project consists of multiple datasets including:
- Transactions
- Customers
- Products
- Orders

## 🔍 Key Steps Performed

### 1. Data Import
- Imported 6 datasets into relational database
- Created structured tables and relationships

### 2. Data Auditing
- Checked for null values
- Identified duplicate records
- Validated data consistency

### 3. Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized formats

### 4. Data Analysis
- Wrote SQL queries to extract insights
- Identified trends and patterns

### 5. Visualization
- Built Power BI dashboard for reporting

## 📈 Key Insights
- Identified sales trends across categories
- Found inconsistencies in transaction data
- Highlighted areas for data quality improvement
